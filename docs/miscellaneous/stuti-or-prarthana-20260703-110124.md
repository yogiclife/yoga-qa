**Original Question:**  
Why stuti/prarthana should not be done whether or not it is done with selfish motive?

**Full Answer:**  
 In what I corrected last week re stuti, my sense was not merely that stuti/prarthana should not be done whether or not it is done with selfish motive because PP knows better what we need than we do. It is certainly true that PP knows better what we need than we do. But even more than that, stuti/prarthana is always done with selfish motive, and for this reason as well it should not be done. There is no element of serving PP with such stuti or prarthana. It is purely 100% selfish; there is no question of "if" it is selfish or not.
