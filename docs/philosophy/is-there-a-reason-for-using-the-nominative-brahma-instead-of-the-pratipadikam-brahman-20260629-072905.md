**Original Question:**  
Is there a reason for using the nominative, Brahma, instead of the pratipadikam, Brahman? It looks like we have been using the pratipadikam for other words, such as atman, Shiva, Krishna, etc.

**Full Answer:**  

Whether we use the nominative or the pratipadikam is not a critical point-- either can be used as we are speaking in English and there are no specific rules as to what form is appropriate when referring to a Samskrta word in an English sentence. But the main point is that it needs to be very clear what word we are talking about. In the case of Brahma if we use the word "Brahman" it will not be clear which Brahman we mean as both the words (pumlinga and napumsakalinga) have the same pratipadikam. For this reason, for Brahma I prefer the nominative form as by our pronunciation (brahma vs brahmA) we can make it very clear which word we mean.
