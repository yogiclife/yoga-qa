**Original Question:**  
Where in srsticakra does the human being's mahattattva get created?

**Full Answer:**  
There remains a fundamental confusion about the krama mahattattva => ahamtattva => citta. This krama occurs only in saincara. In pratisaincara there is no such krama going on, as from the very inception of life at 6:00 the mind of the single celled organism has all three parts of mind present (albeit only citta is active). One of the brothers asked when a jiiva begins to feel it is a separate being-- when it has only mahattattva, or when ahamtattva becomes present? This question is asking about a scenario which does not exist. There is no jiiva which has mahattattva but does not yet have ahamtattva. Any jiiva which has an active mahattattva already had an active ahamtattva even before it had an active mahattattva.
