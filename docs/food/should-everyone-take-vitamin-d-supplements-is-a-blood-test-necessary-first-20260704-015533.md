**Full Answer:**  
Here the point is that there is no one for whom it will not be required. Everyone needs it. Those days are gone when people lived out in the sun. I support getting a blood level checked, but even if one doesn't check the blood level before starting supplementation, one can simply start Vitamin D supplementation-- because everyone's level will be low. 



So one can start taking a daily vegan vitamin D supplement, and then check the level in a few months to see how high it has risen. The level should be around 80 or so. Up to 100 or even slightly above that is fine.
