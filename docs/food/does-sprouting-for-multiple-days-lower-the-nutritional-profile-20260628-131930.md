**Original Question:**  
I had heard, the bigger sprout eats up all nutrition, so it should be just sprouted overnight.

**Full Answer:**  
It is not that the larger sprouts are not nutritious, but that the nutritional profile changes as the sprout grows. In both phases (days 1-2 and days 3-5) it is quite nutritious. The bottom line is: Eat more sprouts! 😊
