**Original Question:**  
Multivitamins as Vitamin D supplement 

**Full Answer:**  
This does contain the B12 one needs, so it is fine for that purpose. I would just suggest going forward that rather than taking a multivitamin, it is better to take just the vitamins you need. This multivitamin -- as with all multivitamins -- contains lots of things which you are already getting plenty of in the fruits and vegetables you eat every day. So you do not need those things. And on the other hand, what is really needed is Vitamin D, which this multivitamin does not contain in sufficient amount. So my recommendation is to get a dedicated vegan B12 supplement and a dedicated vegan Vitamin D3 supplement which contains sufficient Vitamin D. The one I use provides 5000 IU of Vit D3.
