**Original Question:**  
Clarification provided by Acharya ji

**Full Answer:**  
In mukti, unit mind merges with Cosmic Mind, and so no longer remains as a distinct entity. In moksa, unit mind merges into unit consciousness, which merges into Cosmic Consciousness. So in both cases, unit mind merges into something greater and as such no longer remains as unit mind.
